## 🔬 Featured Projects

### 🫁 Embedded Respiratory Breath Classifier
`ESP32-S3` `Embedded ML` `Signal Processing` `Python`

Real-time respiratory breath classification deployed entirely on an ESP32-S3-N1R68 microcontroller with an INMP441 MEMS microphone.
- Audio acquisition via I2S with a state-machine breath segmentation pipeline
- On-device feature extraction: RMS, ZCR, variance, duration
- Multiclass logistic regression model trained on the Respiratory Sound Database, deployed directly to microcontroller
- LED output feedback: Short Breath → Red | Normal → Blue | Long → Green

---

### 🩺 Early Diabetes Prediction — XGBoost ML Pipeline
`Python` `XGBoost` `Scikit-learn` `Feature Engineering` `Medical ML`

14-step end-to-end ML pipeline for early diabetes prediction on the Pima Indians Diabetes Database (768 patients).
- Engineered 8 candidate features (log transforms, interaction terms, ratio features)
- Greedy forward selection via 5-fold cross-validated AUC to prevent overfitting
- XGBoost with RandomizedSearchCV (40 combinations × 5-fold CV), class imbalance correction, L1/L2 regularisation
- False positive error analysis — identified Age as primary misclassification driver; designed and evaluated hard removal vs. soft penalisation correction pipelines
